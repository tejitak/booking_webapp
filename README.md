HashDish WebApp
==============

## Develop
http://hashdish-webapp-dev.herokuapp.com

## Production
http://hashdish.com

## Front-end Dev Setup

### Run npm install
```
npm install
```

### Install and run gulp
```
gulp
```

To minify JS, please run with the additional args
```
gulp --production
```

Access to http://localhost:8080/

## Backend Dev Setup

### Install and run node
```
npm start
```

Access to http://localhost:3000/
